# 《星野方舟 AstraVoxel Ark》游戏设计案

## 1. 一句话概念
一款精致画面的体素沙盒探索建造游戏：玩家在漂浮群岛与星光大陆中采集、建造、种植、探索遗迹，并逐步修复一艘“星野方舟”

## 2. 核心体验
- 像《我的世界》一样自由挖掘、放置方块、建造基地
- 画面更精致：柔和日夜、体积雾、发光植物、水面反射、低多边形角色
- 节奏更轻：少一点硬核生存，多一点探索、收集、装饰、创造
- 适合网页运行，方便别人从 GitHub 打开就玩

## 3. 世界观
玩家醒在一片悬浮大陆上，天空中有碎裂的星环。世界被分成很多生态岛屿：草原岛、樱林岛、晶矿洞、沙海遗迹、雪山浮岛、发光蘑菇地。玩家需要采集不同资源，恢复古老方舟的模块，最终让方舟重新启航

## 4. 视觉风格
### 关键词
精致、治愈、梦幻、低多边形、体素、暖色光照

### 美术方向
- 方块保持体素感，但边缘做轻微倒角或用材质光照增强质感
- 天空盒做成梦幻星云 + 云海
- 草、花、树叶可以是 billboard 或简单 mesh，让画面比纯方块更细腻
- 水面用透明材质 + 法线扰动
- 夜晚加入萤火虫、发光矿石、星光植物

### 参考感
- Minecraft 的自由度
- Dragon Quest Builders 的可爱建造感
- Astroneer 的色彩与探索感
- Zelda: Breath of the Wild 的柔和环境氛围

## 5. 核心玩法循环
1. 探索岛屿
2. 采集方块和特殊资源
3. 合成工具、装饰、设备
4. 建造基地或修复方舟
5. 解锁新岛屿、新材料、新生态
6. 回到探索

## 6. MVP 第一版功能
第一版目标不是做完整游戏，而是做一个能在 GitHub Pages 展示的精致原型

### 必须有
- 可运行网页项目
- 第一人称移动和视角控制
- 程序生成一小片体素地形
- 方块破坏 / 放置
- 基础物品栏
- 至少 8 种方块：草、泥土、石头、木头、树叶、水、晶矿、发光方块
- 日夜循环
- 简单光影和雾效
- README 写清楚怎么运行

### 可以加分
- 小地图或坐标显示
- 简单音效
- 背景音乐
- 截图展示
- GitHub Pages 自动部署

## 7. 方块设计
| 方块 | 用途 | 视觉 |
|---|---|---|
| Grass | 地表 | 绿色顶面 + 土侧面 |
| Dirt | 基础地形 | 温暖棕色 |
| Stone | 矿洞/山体 | 灰色带噪声 |
| Wood | 建造 | 木纹 |
| Leaves | 树冠 | 半透明绿色 |
| Water | 水域 | 半透明蓝色 |
| Crystal Ore | 稀有矿 | 紫蓝发光点 |
| Glow Block | 装饰/照明 | 柔和发光 |

## 8. 地图生成
MVP 使用简单高度图：
- Perlin/Simplex noise 生成高度
- 高度低处放水
- 地表草，下面泥土，再下面石头
- 随机生成小树和晶矿点

后续版本再做：
- 区块加载 chunk system
- 洞穴生成
- 不同生态群系
- 浮岛地形

## 10. 技术建议
### 推荐路线 A：Three.js
优点：轻、自由、适合 GitHub Pages

建议结构：
```text
src/
  main.ts
  game/Game.ts
  world/World.ts
  world/Chunk.ts
  world/BlockRegistry.ts
  player/PlayerController.ts
  render/Materials.ts
  ui/Hotbar.ts
```

### 推荐路线 B：Babylon.js
优点：内置工具更多，物理、相机、材质方便

如果目标是快速做精致画面，Babylon.js 也很合适
